# TP_Bibliothèque_POO_Structure
Ce dépôt contient le projet "TP Bibliothèque" réalisé dans le cadre d'un cours de Programmation Orientée Objet et PHP. Le contenu du dépôt correspond exactement aux fichiers demandés pour le rendu du travail.

## Important — Pas de `.gitignore` ni d'exclusions
je n'ai pas ajouté de fichier `.gitignore` ni exclu de fichiers du dépôt. Tous les fichiers sources, scripts SQL et fichiers de configuration nécessaires au fonctionnement et à la correction du TP sont inclus ici afin que l'enseignant puisse récupérer l'intégralité du projet sans omission.

## Exécution locale (XAMPP)
1. Placer le dossier `TP_Bibliotheque_POO_Structure` dans le dossier `htdocs` de XAMPP 
2. Démarrer Apache et MySQL via le panneau XAMPP.
3. Importer `structure.sql` dans votre instance MySQL (par phpMyAdmin ou ligne de commande) pour créer la base et les tables.
4. Vérifier ou ajuster les paramètres de connexion dans `db.php` (hôte, utilisateur, mot de passe, nom de la base).
5. Ouvrir dans le navigateur : `http://localhost/TP_Bibliotheque_POO_Structure/`
